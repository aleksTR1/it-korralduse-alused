# Teema 4: Lean

## Põhimõisted

* Lean – pideva parendamise meetod.
* Väärtus – kliendile oluline tegevus.
* Raiskamine – tegevus, mis ei loo väärtust.
* Kaizen – pidev areng.
* Tõhusus – töö parem korraldus.

## Mida õppisin

Õppisin Lean põhimõtteid. Sain aru, kuidas vähendada raiskamist. Mõistsin pideva arendamise tähtsust. Tutvusin Kaizeni mõistega. Sain teadmisi tõhusast töökorraldusest.

## Näide

Kui sama tööd tehakse mitmes kohas topelt, aitab Lean protsessi lihtsustada.
