# Teema 3: Taristu

## Põhimõisted

* Server – teenuseid pakkuv arvuti.
* Võrk – seadmete ühendus.
* Ruuter – liikluse suunamise seade.
* Lüliti – võrguühenduste haldur.
* Andmekeskus – serverite asukoht.

## Taristu tabel

| Seade    | Ülesanne           |
| -------- | ------------------ |
| Server   | Teenuste pakkumine |
| Ruuter   | Liikluse suunamine |
| Lüliti   | Võrgu ühendamine   |
| Tulemüür | Turvalisus         |
| UPS      | Varutoide          |

## Mida õppisin

Õppisin IT taristu peamisi komponente. Sain aru serverite rollist. Mõistsin võrgu toimimise põhimõtteid. Tutvusin ruuterite ja lülitite tööga. Sain ülevaate andmekeskuste tähtsusest.

## Näide

Ettevõtte veebileht töötab serveris, mis asub andmekeskuses.
