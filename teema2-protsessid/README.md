# Teema 2: Protsessid

## Põhimõisted

* Protsess – tegevuste kogum eesmärgi saavutamiseks.
* Sisend – protsessi algandmed.
* Väljund – protsessi tulemus.
* KPI – tulemuslikkuse näitaja.
* Dokumentatsioon – protsessi kirjeldus.

## Mõistete tabel

| Mõiste          | Selgitus           |
| --------------- | ------------------ |
| KPI             | Tulemusnäitaja     |
| Sisend          | Alginfo            |
| Väljund         | Tulemus            |
| Protsess        | Tegevuste jada     |
| Dokumentatsioon | Kirjalik kirjeldus |

## Mida õppisin

Õppisin, kuidas organisatsioonid kasutavad protsesse töö korraldamiseks. Sain aru, et protsessid aitavad tööd standardiseerida. Mõistsin KPI tähtsust. Tutvusin sisendite ja väljundite mõistega. Sain teadmisi dokumenteerimise kohta.

## Näide

Uue töötaja lisamine ettevõtte süsteemidesse toimub kindla protsessi järgi.
