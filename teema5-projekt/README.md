# Teema 5: Projektijuhtimine

## Põhimõisted

* Projekt – ajutine tegevus eesmärgi saavutamiseks.
* Projektijuht – projekti eest vastutaja.
* Eesmärk – soovitud tulemus.
* Risk – võimalik probleem.
* Tähtaeg – projekti lõppkuupäev.

## Riskide tabel

| Risk               | Mõju                |
| ------------------ | ------------------- |
| Tähtaja ületamine  | Projekti hilinemine |
| Eelarve kasv       | Suurem kulu         |
| Tehniline probleem | Töö seiskumine      |
| Ressursipuudus     | Aeglasem töö        |
| Vale planeerimine  | Kehvem tulemus      |

## Mida õppisin

Õppisin projektijuhtimise põhitõdesid. Sain aru eesmärkide tähtsusest. Mõistsin riskide mõju projektidele. Tutvusin projektijuhi ülesannetega. Sain teadmisi planeerimisest.

## Näide

Veebilehe loomine ettevõttele on projekt, millel on eesmärk, tähtaeg ja eelarve.
